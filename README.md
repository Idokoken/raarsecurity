#  RAAR Security ltd Website

## Description

RAARsecrutyltd is a professional website designed to showcase the services offered by RaaRsecrutyltd, a company dedicated to providing security services to their clientele. The website provides detailed information about our services, contact details, and a user-friendly interface to engage customers, partners, and stakeholders. Built with modern web technologies, the site is optimized for accessibility, responsiveness, and search engine visibility.

## Technology used

The technologies used for developing the website includes
1. React(Javascript)
2. CSS
3. Bootstrap Framework
4. Some external dependencies like styled-component, react-router-dom etc.


The hosted(live) website can be viewed on **[https://raarsecurityltd.com](https://raarsecurityltd.com)**.
